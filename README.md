# PET DAY
​
## サイト概要
### サイトテーマ
自分のペットの思い出を記録し共有し合えるSNS
​
### テーマを選んだ理由
実家にペットがいましたが離れて暮らさなければいけない状況になり、ペットにたしての記録が少なく寂しさを感じていました。
他にも自分と同じような境遇の人がいて思い出を共有し合えたらと考えました。また、動物好きな人たちの交流の場になれればと考えこのSNSを作りました。

### ターゲットユーザ
- ペットに興味がある人
- ペットと離れて暮らしている人
- 動物が好きな人
​
### 主な利用シーン
- 出来事を記録する時
- いい写真が撮れた時
